# experiments/ — доменные прогоны (требуют закрытых данных/солвера)

Эти скрипты воспроизводят доменные результаты статьи. В отличие от
самодостаточного ядра (`../laziness.py`, `../baselines.py`, `../synthetic_demo.py`,
`../sinkhorn_demo.py`, работающих на одном `numpy`), они **зависят** от пакетов
`python_forecast` / `python_solver` и от данных Росстата (рынок труда), которые в
публичный репозиторий не входят. Здесь — для провенанса (точный использованный код).

- `explore_laziness.py` — главный прогон лени (синтетика + энергодиагностика +
  rolling-origin sweep по λ для null/ewma; ОКВЭД-1/ОКВЭД-2). → `docs/laziness_experiments.md`.
- `explore_baselines.py` — сравнение с baseline'ами: raw / ewma (временная
  гладкость, brand-share) / null (наш fit-exact), с колонкой невязки баланса
  `||A·Xs−A·X||/||A·X||` + энергией; плюс Sinkhorn-демо. → раздел «Baselines».
- `econ_significance.py` — тесты значимости для рынка труда (ОКВЭД-2): Дибольд—
  Мариано (HLN), Уилкоксон по пулу поиндикаторных |ошибок|, Пезаран—Тиммерман,
  кластерный бутстрап; плюс сильные одномерные эталоны (блуждание с дрейфом, AR(1),
  Хольт-ETS). Главный честный вывод: ETS — сильнейший точечный прогноз; сдвиг
  приора по надёжности незначим (3 момента). → §5.1, табл. «Сильные эталоны».
- `physics_relaxation_rate.py` — спектральный зазор γ=1−ρ* восстановленного
  огрублённого (Улам) оператора против аналитической скорости сжатия Ландау—
  Лифшица: Λ_восст≈0.68 vs Λ=0.72 (отношение 0.95). → §5.2 / §6, табл. физики.
  Требует `MultiClusterPhysics/` (rapidity_ensemble) + `python_solver`. Зависит от
  `scipy` (econ_significance) — `pip install scipy`.
- `timing.log` — журнал времени прогонов (CPU; GPU не требуется).

Канонические версии живут в `FindProbability/python_forecast/scripts/`. Запуск
(из `FindProbability/`):

```bash
python -m python_forecast.scripts.explore_laziness
python -m python_forecast.scripts.explore_baselines
```

Для публичного репозитория планируется заменить чтение закрытых данных на
синтетический генератор или приложить обезличенные ряды `P̃` (CSV).
